# 🥼 Dietas.py – Automação de Baixa de Itens no Sistema IW

## 📋 Descrição
Este projeto é uma automação desenvolvida em Python para otimizar o processo de baixa de itens no sistema IW, utilizado na área hospitalar.  
O programa interage com a interface gráfica, reduzindo erros manuais e aumentando a eficiência dos processos internos.

## 🛠️ Tecnologias Utilizadas
- Python
- PyAutoGUI
- PyScreeze
- Pyperclip
- Time
- OS

## 🚀 Funcionalidades
- Localização automática de botões e campos na tela por reconhecimento de imagem.
- Geração de etiquetas automáticas a partir de código de barras e quantidade.
- Preenchimento automatizado de campos de login e senhas.
- Automação do protocolo de entrega (opcional).
- Validação de entradas do usuário para maior segurança.

## 🎯 Principais Desafios
- Implementação de tentativas múltiplas para localizar elementos da interface em caso de falhas.
- Sincronização com os tempos de carregamento do sistema.
- Design de interação simples e amigável com o usuário final.

## 📈 Resultados
- Redução do tempo para baixa e protocolo de itens.
- Diminuição de erros operacionais.
- Aumento da produtividade nos processos de dispensação de dietas hospitalares.

## ⚡ Como utilizar
> *Observação*: Para rodar o script é necessário configurar corretamente o diretório das imagens e ter as bibliotecas instaladas.

1. Instale as bibliotecas necessárias:
    ```bash
    pip install pyautogui pyscreeze pyperclip
    ```

2. Execute o arquivo `Dietas.py` e siga as instruções no terminal.
